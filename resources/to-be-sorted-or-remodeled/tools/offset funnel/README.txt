# Summary

A simple offset funnel for those occasions when the wide big gets in the way. Drawn in FreeCAD so that when you need exactly THAT size the shop ain't got or it's a Sunday and you can't get it

# Print Settings

Printer Brand: RepRap
Printer: Homebrew
Rafts: Doesn't Matter
Supports: No

Notes: 
With the nozzle and layer height, I'd go big and save yourself a load of time.  Add a brim or it might pop off the printbed.  Since the wall thickness is 1.5mm try a 0.6mm nozzle if you have it.  I printed mine with 0.4mm nozzle and a layer height of 0.3mm.
